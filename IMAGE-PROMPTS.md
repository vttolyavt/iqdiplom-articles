# Промпты для генерации изображений к статьям

## Общие правила
- Формат: 16:9 (1200x675) для статей, 1:1 для превью
- Стиль: плоская иллюстрация, минимализм, профессиональный
- Цвета: синий (#2563eb), фиолетовый (#764ba2), белый
- Сохранять как: WebP, качество 80%, имя = slug статьи
- Папка: `articles/images/`

## Технические требования к изображениям

### Форматы и размеры:
- Hero изображения: 1200x675px (16:9), WebP
- Инфографики: 800x1200px (2:3 вертикаль) или 1200x800px (3:2 горизонт), WebP/PNG
- Иконки/аватары: 400x400px (1:1), WebP
- CTA баннеры: 1200x400px, WebP

### Оптимизация:
- WebP формат, качество 80%
- Максимальный размер файла: 150KB для hero, 200KB для инфографик
- Alt-текст на русском для каждого изображения
- Lazy loading: добавить `loading="lazy"` к тегам img

### HTML вставка:
```html
<figure>
    <img src="images/vvedenie-hero.webp"
         alt="Как написать введение к дипломной работе — пошаговое руководство"
         width="1200"
         height="675"
         loading="lazy">
    <figcaption>Структура введения к дипломной работе включает 6 обязательных элементов</figcaption>
</figure>
```

### Создание папки:
```bash
mkdir articles/images
```
